# BOEF v0.1.0-public

Initial public source release of BOEF.

## What Is Included

- PySide6 desktop application source.
- BOEF engineering core for railway track analysis and beam-on-elastic-foundation workflows.
- Database models, migrations, seed logic, and project import/export helpers.
- Test suite and regression fixtures.
- Engineering reference documentation.
- MIT licence, attribution notice, citation metadata, contribution guidance, and security guidance.
- Public agent instructions in `AGENTS.md` and `python-app/AGENTS.md`.
